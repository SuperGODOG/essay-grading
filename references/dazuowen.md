# 大作文 - 写作指导与批改标准

> 此文件由 SKILL.md 在识别为「大作文」后加载。
> ⚠️ 当前版本：具备完整写作指导，批改评分细则为通用标准，待细化。

## 核心写作原则

### 基本要求
1. **文体定位**：申论作文是议论文，不是对策文或策论文
2. **形式重于内容**：在应试环境下，正确的格式和结构比内容创新更重要
3. **完整性要求**：文章必须有完整的开头、主体、结尾结构
4. **字数控制**：严格控制在要求字数范围内，通常800-1000字
5. **时效性处理**：避免使用时效性强的词汇，如具体节日、年份等

### 评分导向
- **高分策略**：形式规范、结构清晰、论证充分
- **争位次策略**：在保证形式的基础上，注重内容的深度和创新

## 标题拟定技巧

**铁律**：标题中必须包含题干给出的关键主题词，这是硬性要求，不可违背。

### 标题类型与方法

1. **直接引用法**：适用于全命题作文，直接使用题干中的关键句子
2. **短缩法**：将长句浓缩成短标题，提取核心概念
3. **对仗结构法**：使用工整的对仗句式，结构：动词+主题词+愿景
4. **关键词组合法**：结合材料中的关键词，保持题干主题词不变

### 标题写作技巧
1. 避免副标题：不要自己添加副标题
2. 长度适中：一般控制在15字以内
3. 语言简洁：用词精准，避免冗余
4. 朗朗上口：读起来顺畅，有节奏感

## 开头段写作技巧

### 四种开头方式

1. **直接解释题干法（最简单）**：直接解释题干含义 → 引用相关政治理论 → 点明总论点
2. **政治背景开头法（最稳妥）**：引用二十大报告等政治背景 → 结合材料关键词 → 过渡到总论点
3. **哲学背景开头法**：引用马克思主义哲学原理 → 解释现实意义 → 连接到具体主题
4. **现实背景开头法**：描述当前形势 → 指出问题或机遇 → 提出总论点

### 开头段注意事项
- 长度控制：一般3-4句话
- 避免空洞：不要写过于宽泛的套话
- 自然过渡：从背景到总论点要自然衔接
- 关键词呼应：确保包含题干关键词

## 过渡段写作技巧

**核心原则**：必须与开头重复至少一个关键概念，但要用不同句式表达。

**三步走结构**：
1. 抄题目要求：直接引用题目中的关键句
2. 解释核心概念：阐释关键词含义
3. 点题过渡：自然过渡到分论点

**注意事项**：
- 点而不论：提及总论点关键词但不展开论述
- 避免重复：与开头段内容有呼应但不完全重复
- 长度适中：一般2-3句话

## 分论点拟定技巧

### 四种方法

1. **拆题法（最直接）**：直接拆解题目中的关键词
2. **就近法（最省事）**：直接使用材料中的并列结构
3. **概括法（最常用）**：用普通词汇总结材料大意
   - 思想层面：理念、意识、观念
   - 制度层面：机制、体系、制度
   - 实践层面：行动、措施、实践
4. **补充法（最后选择）**：实在找不到第三个分论点时自己编一个

### 分论点写作要求

**必须包含总论点的关键词**：这是分论点写作的铁律。

**长分论点四步公式**：
1. 总论点主题词
2. 分论点关键词
3. 4+4结构解释
4. 愿景式结尾

### 分论点写作禁忌
1. 分论点不明确：表述模糊
2. 分论点重复：不同分论点表达相同内容
3. 总分论点雷同：分论点与总论点过于相似
4. 双要素重复：避免A+B+A+B的重复结构

## 分论点论述段写作

**基本结构**：
1. 分论点句：明确提出分论点
2. 理论阐释：解释分论点的重要性
3. 案例论证：使用具体例子支撑
4. 对策建议：适度展开解决方案
5. 小结句：呼应分论点

**案例使用原则**：
1. 优先使用材料案例
2. 经典案例备选：共和国勋章获得者事迹、重大历史事件、典型改革实践
3. 避免偏门案例
4. 案例要"圆"回来：确保案例能支撑分论点

**对策建议写作**：
- 适度展开，不要写成详细的对策方案
- 紧扣主题
- 语言简洁

## 结尾段写作技巧

### 结尾段的作用
1. 收束全文：对全文进行总结
2. 升华主题：提升文章立意
3. 呼应开头：与开头形成呼应

### 结尾段写作方法

1. **套路化结尾（推荐）**：开头段总论点 + 中间段分论点 + 总论点关键词收尾
2. **愿景式结尾**：展望美好未来，使用积极向上的词汇

**注意事项**：
- 避免新观点：不要在结尾提出新的论点
- 语言有力：用词要有感染力和号召力
- 长度适中：一般2-3句话

## 语言表达技巧

### 政治语言运用
- "新发展理念"（创新、协调、绿色、开放、共享）
- "中国式现代化"
- "高质量发展"
- "坚持以人民为中心"
- "统筹发展和安全"
- "推进国家治理体系和治理能力现代化"

### 句式运用技巧
1. 对仗句式：增强语言的节奏感
2. 排比句式：增强语言的气势
3. 设问句式：引起读者思考
4. 感叹句式：表达强烈情感

## 常见问题与解决方案

| 问题 | 解决方案 |
|------|---------|
| 材料数据是否需要写入？ | 不需要写具体数据，重点是理解数据背后的含义 |
| 如何处理时效性强的内容？ | 将具体时间替换为通用表述，如"端午节"→"传统节日" |
| 分论点找不到第三个？ | 优先考虑两个分论点结构；如必须三个，用补充法 |
| 案例不够用？ | 深挖材料案例；使用经典案例；学会"圆"案例 |
| 字数不够或超标？ | 不够：展开对策建议；超标：删减案例描述 |

## 应试策略

### 时间分配建议
- 审题立意：5-8分钟
- 拟定提纲：3-5分钟
- 写作时间：35-40分钟
- 检查修改：2-3分钟

### 得分策略
1. **保底策略**：确保格式正确、结构完整
2. **提升策略**：在保证形式的基础上优化内容
3. **冲刺策略**：使用高质量的政治语言和深刻的分析

## 批改评分细则（通用标准版）

> ⚠️ TODO: 以下为基于通用框架的大作文评分标准，待根据实际批改需求细化。

### 评分权重分配
- **内容评分（70%）**: 总论点明确性、分论点支撑度、论证充分性、立意深度
- **结构评分（10%）**: 开头-主体-结尾完整性、段落过渡、逻辑递进
- **语言评分（20%）**: 政治语言运用、句式丰富度、语法规范性

### 打分执行标准

每次批改必须包含：
1. 分项得分明细：内容得分（含总论点/分论点/论证子项）、结构得分、语言得分
2. 扣分原因说明
3. 最终分数计算（公式：内容得分×0.7 + 结构得分×0.1 + 语言得分×0.2）
4. 等级评定（一类文/二类文/三类文/四类文）
5. 针对性建议：至少3条改进措施

### 要点评分侧重

**总论点（30%权重）**：
- 是否明确、是否扣题、是否有深度

**分论点（40%权重）**：
- 数量是否合理（根据材料特点灵活调整，反对机械套用"五段三分"）
- 是否包含总论点关键词
- 是否相互独立、不重复
- 是否支撑总论点

**论证（30%权重）**：
- 是否使用材料案例
- 理论阐释是否到位
- 对策建议是否适度

### 结构评分侧重
- 开头是否包含政治背景/题干解释 + 总论点
- 过渡段是否自然衔接
- 分论段是否有前置提炼
- 结尾是否呼应开头、升华主题

### 语言评分侧重
- 政治术语使用是否恰当
- 句式是否丰富（对仗、排比、设问等）
- 是否有语病或冗余

### 等级评定
| 等级 | 得分率 | 说明 |
|------|--------|------|
| 一类文 | 85%+ | 论点明确、论证充分、结构完整、语言流畅 |
| 二类文 | 70-84% | 论点较明确、论证较充分、结构较完整 |
| 三类文 | 60-69% | 论点基本明确、论证基本充分 |
| 四类文 | <60% | 偏离主题、结构混乱、论证不足 |

## 打分示例模板

```
【题目】"给定资料5"中提到"流动与新生"，请深入思考这句话，自选角度，自拟题目，写一篇文章。（40分，800-1000字）

【考生文章】[考生文章内容]

【批改评分】
✅ 内容得分：24/40分
- 总论点（30%）："在流动中实现新生"观点明确，扣题 ✓（10分）
- 分论点（40%）：两个分论点支撑总论点
  - 分论点1："要素流动激发经济活力"论证较充分 ✓（8分）
  - 分论点2："人才流动促进社会进步"论证稍弱（-4分）
- 论证（30%）：使用材料案例较少，缺少理论阐释（-6分）

✅ 结构得分：6/10分
- 开头：政治背景开头，自然引出总论点 ✓（2分）
- 主体：分论段结构完整，但缺少过渡段（-1分）
- 结尾：呼应开头，语言有号召力 ✓（2分）

✅ 语言得分：10/20分
- 政治语言运用：个别术语使用不当（-3分）
- 句式丰富度：以陈述句为主，缺少变化（-4分）
- 规范性：有2处语病（-3分）

【最终得分】：24×0.7 + 6×0.1 + 10×0.2 = 19.4分
  （内容24×0.7=16.8 + 结构6×0.1=0.6 + 语言10×0.2=2.0）
【等级评定】：不及格（48.5%）

【改进建议】：
1. 强化分论点2论证：补充具体案例和理论分析
2. 增加材料案例：将材料中的具体案例融入论证
3. 丰富句式：增加对仗、排比句式增强气势
4. 添加过渡段：在开头和分论段之间增加过渡
```
