# 综合分析题 - 批改标准与答题框架

> 此文件由 SKILL.md 在识别为「综合分析题」后加载。

## 答题技巧总结

- **150字题目**: 建议不分段
- **300字题目**: 建议分段作答
- **要点全面性**: 重于形式完美性
- **语言简洁性**: 避免冗长描述
- **逻辑一致性**: 保持要点间逻辑关系
- **数字处理原则**: 具体数据类材料尽量概括作答
- **要点合并核心技能**:
  - 从材料中提取的要点数 ≠ 最终答题条数
  - 相同性质要点必须合并
  - 相关联要点必须整合
  - 按逻辑关系组织条目（时间、空间、重要性、因果关系等）

## 字数-结构对应表

```
字数限制 | 综合分析结构 | Why部分要点数 | 合并比例
--------|-------------|-------------|----------
150字   | What1+Why2-3+How1 | 2-3个分析点，合并为1-2条 | 60-80%
200字   | What1+Why3-4+How1 | 3-4个分析点，合并为2条 | 60-80%
250字   | What1+Why4-5+How1 | 4-5个分析点，合并为2-3条 | 60-80%
300字   | What1-2+Why4-5+How1 | 4-5个分析点，合并为2-3条 | 60-80%
350字   | What1-2+Why5-6+How1 | 5-6个分析点，合并为3条 | 60-80%
400字   | What1-2+Why6-7+How1 | 6-7个分析点，合并为3-4条 | 60-80%
450字   | What1-2+Why7-8+How1 | 7-8个分析点，合并为4条 | 60-80%

⚠️ 综合分析题结构相对固定，不完全按字数计算条数
Why部分的材料要点数 ≈ 字数×1/25×1/2×0.7，但需合并呈现
```

## 题干要求关键字暗示

- 观点明确: 第一句对作答对象字面描述
- 逻辑性: 按照时间叙事或者方面叙事（如经济、社会、文化、生态等方面）
- 注意作答对象的积极负面性: 负面则作答偏负面信息；正面则多找正面案例

## 题型核心特征

**题型识别**:
- 对象抽象性：解读概念/现象/观点（如"种种遮蔽""预先失败"）
- 动词标志："分析""评述""阐释""理解"

**答题步骤**:
- 提取要点：从材料中筛选与对象相关的所有信息（避免主观添加，忽略具体数字）
- 分层归类：按"现象解释→原因/影响→结论"逻辑排序
- 答题结构：紧扣字数，突出重点。概括为"中好坏策"
  1. '中': 解释含义（1-2句）
  2. '好坏': 分析正反影响（根据作答主体的积极消极性调整好坏比例）
  3. '策': 结论对策（1句收尾）

**3W逻辑深化框架**:
- **What**: 用材料原文定义本质（忌主观改写，忌抄数字）
- **Why**: 正反论证需引用材料矛盾点
  - 影响分析按"经济→社会→文化"维度排序
  - 示例：文艺"遮蔽"既保护传统又阻碍创新
- **How**: 对策篇幅≤分析部分的1/3

**高频失分点**:
- ✘ 误将"分析原因"归入此类（实为概括题）
- ✘ 脱离材料空谈对策
- ✘ 混淆"综合分析"与"应用文"（后者3W中"怎么办"可能是重点）
- ✘ 抄写材料中的具体数字数据

## 批改评分细则

### 评分权重分配
- **内容评分（70%）**: 检查主题围绕度、观点清晰度、分析深度
- **结构评分（10%）**: 分析答题结构完整性、段落过渡自然性
- **语言评分（20%）**: 标注语病、冗余表述，给出修改建议

### 打分执行标准

每次批改必须包含：
1. 分项得分明细：要点得分、表述得分、格式得分
2. 扣分原因说明：每个扣分点明确理由
3. 最终分数计算：展示完整计算过程
4. 等级评定：根据得分率
5. 针对性建议：至少3条改进措施
6. 结构完整性检查：必须检查What-Why-How三部分是否完整

### 各维度评分细则

**要点评分**:
- 完全正确且表述准确：满分
- 要点正确但表述不够准确：扣20-30%
- 要点部分正确：扣40-60%
- 要点错误或遗漏：0分

**表述评分**:
- 语言准确性（40%）：术语使用、表达精准度
- 条理性（40%）：逻辑清晰、层次分明
- 规范性（20%）：语法正确、无语病

**格式评分**:
- 字数控制（50%）：严格按题目要求
- 段落安排（30%）：根据字数要求合理分段
- 标点使用（20%）：标点符号使用规范

## 打分示例模板

```
【题目】请分析"预先失败"这一现象。（20分，不超过300字）

【考生答案】[考生作答内容]

【批改评分】
✅ 要点得分：12/20分
- What部分：准确定义"预先失败"现象 ✓（4分）
- Why部分：分析原因和影响 ✓（6分）
  - 心理因素分析 ✓
  - 社会环境影响 ✓
  - 缺少制度保障分析 ✗（-2分，未深入分析）
- How部分：提出解决思路 ✓（2分）

✅ 结构完整性检查：What-Why-How结构完整 ✓
- What部分：1句话定义，符合要求
- Why部分：2条分析，合并适当
- How部分：1句结论，篇幅合理

✅ 表述得分：8/20分
- 语言准确性：较好，个别表述不够精准（-4分）
- 条理性：逻辑清晰，层次分明 ✓
- 规范性：基本规范，有1处语病（-4分）

✅ 格式得分：16/20分
- 字数控制：295字，符合要求 ✓
- 段落安排：分段合理 ✓
- 标点使用：基本正确，有1处错误（-4分）

【最终得分】：(12/20×0.7 + 8/20×0.2 + 16/20×0.1) × 20 = 13.2分
【等级评定】：良好（66%）

【改进建议】：
1. 深化Why部分分析：补充制度保障缺失对"预先失败"的影响
2. 优化表述：将"心理问题"改为"心理预期偏差"
3. 注意语法：检查主谓搭配，避免语病
4. 数字处理：如有具体数字数据，建议删除或改为概括性表述
5. 逻辑深化：在Why部分体现辩证思维
6. 材料结合：更紧密结合材料中的具体案例
```
