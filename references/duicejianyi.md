# 对策建议题 - 批改标准与答题框架

> 此文件由 SKILL.md 在识别为「对策建议题」后加载。

## 答题技巧总结

- **150字题目**: 建议不分段
- **300字题目**: 建议分段作答
- **要点全面性**: 重于形式完美性
- **语言简洁性**: 避免冗长描述
- **逻辑一致性**: 保持要点间逻辑关系
- **数字处理原则**: 具体数据类材料尽量概括作答
- **要点合并核心技能**:
  - 从材料中提取的要点数 ≠ 最终答题条数
  - 相同性质要点必须合并
  - 相关联要点必须整合
  - 按逻辑关系组织条目

## 字数-答题条数对应表

```
字数限制 | 对策题条数 | 材料要点数 | 合并比例
--------|----------|----------|----------
150字   | 2±1条    | 2±1个    | 60-80%
200字   | 2-3条    | 3±1个    | 60-80%
250字   | 3-4条    | 4±1个    | 60-80%
300字   | 3-5条    | 5±1个    | 60-80%
350字   | 4-5条    | 6±1个    | 60-80%
400字   | 4-6条    | 7±1个    | 60-80%
450字   | 5-7条    | 8±1个    | 60-80%

注意：对策题每个条目篇幅更长（3-4行），总条数比概括题少
```

## 题干要求关键字暗示

- 要求针对性: 需写出问题描述
- 要求可行性: 需要适当细节描述（但不必描述具体数据）

## 题型核心特征

**审题定题型**:
- 动词识别：题目中出现"提出建议""拟定措施""说明对策"等动词即为对策建议题
- 对象锁定：
  - 若对象是具体问题（如"为促进咖啡产业发展提建议"），需针对材料问题直接提对策
  - 若对象是任务目标（如"拟定脱贫攻坚主要任务"），需结合应用文格式（如分任务+措施）
- 题型区分：
  - 对策建议题：以建议/对策为核心（占答案80%以上篇幅）
  - 综合分析题：对策仅作为结论（How环节）出现

**材料要点提取与答题条数规划**:
```
第一步：材料要点提取数 ≈ 概括题材料要点数 × 0.8
• 300字限制 ≈ 5±1个材料要点（需合并为3-5条对策）
• 200字限制 ≈ 3±1个材料要点（需合并为2-3条对策）

第二步：对策条数整合原则
• 同类对策合并（如"加强宣传"+"提高意识"="强化宣传教育"）
• 按实施主体分类（政府、企业、社会、个人）
• 按实施层面分类（制度、管理、技术、文化）
```

**对策来源**:
- 材料有直接对策：优先提取尾段明确提出的对策（中间段落对策需谨慎判断）
- 材料无直接对策：不可反推对策（如"资金不足"不能自行写"加大资金投入"），仅用一句话号召性结尾

**对策针对性**: 每条对策必须直接回应材料问题（如问题为"道路破损"，对策需具体到"修缮村内主干道"而非泛泛而谈"改善基础设施"）

**常见陷阱**:
- 题目要求"提建议"，但材料中已包含完整对策 → 需直接提取，不可修改
- 问题描述模糊（如"如何解决XX困境"）→ 需先明确材料中的具体问题再提对策
- 反推对策（如"资金不足"→"加大资金投入"）→ 严重错误

**对策篇幅控制**:
- 问题部分需精简（≤4行），对策占主体（约8行）
- 每条对策需有前置关键词（如"完善制度""加强宣传"），体现逻辑层次

## 批改评分细则

### 评分权重分配
- **内容评分（70%）**: 检查主题围绕度、观点清晰度、对策具体性
- **结构评分（10%）**: 分析答题结构完整性、段落过渡自然性
- **语言评分（20%）**: 标注语病、冗余表述，给出修改建议

### 打分执行标准

每次批改必须包含：
1. 分项得分明细：要点得分、表述得分、格式得分
2. 扣分原因说明：每个扣分点明确理由
3. 最终分数计算：展示完整计算过程
4. 等级评定：根据得分率
5. 针对性建议：至少3条改进措施
6. 要点数量检查：材料要点提取数和最终答题条数是否合理

### 各维度评分细则

**要点评分**:
- 完全正确且表述准确：满分
- 要点正确但表述不够准确：扣20-30%
- 要点部分正确：扣40-60%
- 要点错误或遗漏：0分

**表述评分**:
- 语言准确性（40%）：术语使用、表达精准度
- 条理性（40%）：逻辑清晰、层次分明
- 规范性（20%）：语法正确、无语病

**格式评分**:
- 字数控制（50%）：严格按题目要求
- 段落安排（30%）：根据字数要求合理分段
- 标点使用（20%）：标点符号使用规范

```
【题目】针对材料中反映的问题，提出解决建议。（15分，不超过300字）

【考生答案】[考生作答内容]

【批改评分】

- 对策1：完善监管制度 ✓
- 对策2：加强宣传教育 ✓

- 对策5：加大资金投入 ✗（-3分，未提及）

✅ 要点数量检查：300字题目应提取5±1个材料要点，合并为3-5条对策，实际3条，数量合理 ✓

✅ 表述得分：6/15分
- 语言准确性：较好，个别表述不够精准（-3分）
- 条理性：有前置关键词，逻辑清晰 ✓
- 规范性：基本规范，有2处语病（-3分）

✅ 格式得分：12/15分
- 字数控制：295字，符合要求 ✓
- 段落安排：分段合理 ✓
- 标点使用：基本正确，有1处错误（-3分）

【最终得分】：(9/15×0.7 + 6/15×0.2 + 12/15×0.1) × 15 = 10.65分
【等级评定】：良好（71%）

【改进建议】：
1. 补充遗漏对策：建立长效机制、加大资金投入等关键措施
2. 优化表述：将"加强管理"改为"完善监管执法体系"
3. 注意语法：检查主谓搭配，避免语病
4. 数字处理：如有具体数字数据，建议删除或改为概括性表述
5. 对策针对性：每条对策应直接回应材料中的具体问题
6. 逻辑组织：按实施主体或层面排序
```
