## 二、优势

| 优势 | 体现 |
|------|------|
| **题型识别极稳** | 四种不同题型全部一次命中，无需人工干预 |
| **Progressive Disclosure 高效** | 每次仅加载对应 reference，其他 4 个文件不占 context |
| **3W 框架一致性强** | 综合分析题严格遵循 What-Why-How，应用文遵循背景-现状-问题-建议 |
| **字数管理精确** | 4 题均控制在限字 ±10 字以内（198/200、340/350、490/500、920/1000） |
| **数字过滤彻底** | 标准答案中无任何材料具体数字残留 |
| **材料利用度高** | 每题均标注了答案来源说明，覆盖所有材料段落 |
| **混合题型正确处理** | 第 3 题自动应用了 1:2 问题/对策字数分配规则 |
| **"无作答"路径正确** | 4 题均跳过批改评分，直接输出标准答案 |
